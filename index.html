Как использовать:

  1. Скачайте ваш проект из Scratch (Файл → Сохранить на компьютер)
  2. Загрузите .sb3 файл в этот конвертер
  3. Настройте параметры (название, цвета и т.д.)
  4. Нажмите "Конвертировать в HTML"
  5. Скачайте HTML файл и загрузите на ваш сайт

Примечание: Сгенерированный HTML файл содержит весь проект внутри себя и не требует интернета для работы.
